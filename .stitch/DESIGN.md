# RecoverAI Design System

## 1. Brand Guidelines
RecoverAI represents professional spinal care. The aesthetics should feel medical yet modern, trustworthy, and precise.

## 2. Typography
- **Primary/Display**: Lato (sans-serif)
- **Secondary/Body**: Merriweather (serif)

## 3. Color Palette
- **Primary**: Deep Black (`#1a1a1a`)
- **Secondary**: Vivid Red (`#e63b2e`)
- **Accent**: Electric Lime (`#C0F84B`)
- **Background**: Soft Bone/Warm White (`#f5f0e8`)
- **Surface**: Pure White (`#FFFFFF`)

## 4. Visual Elements
- Cards with subtle blur and transparency (Glassmorphism where appropriate).
- Deep shadows for elevation (`shadow-2xl` on heroes and key elements).
- Rounded corners (`rounded-[3rem]`, `rounded-full` for buttons).

## 5. Components
- **Buttons**: Pill-shaped or large rounded rects with black text/background. Hover states translate Y slightly up.
- **Badges**: Small uppercase tracking-widest text with a colored background.

## 6. Design System Notes for Stitch Generation
**DESIGN SYSTEM (REQUIRED):**
Use Tailwind CSS.
Colors:
- `surface`: `#f5f0e8`
- `primary`: `#1a1a1a`
- `secondary`: `#e63b2e`
- `electric-lime`: `#C0F84B`
- `on-surface-variant`: `#4a4a4a`
- `surface-container-low`: `#f2ede5`
- `surface-container`: `#eee9e0`
Typography:
- Use `font-display` (Lato) for headings.
- Use `font-body` (Merriweather) for body text.
Style: Professional, medical, rounded-3xl containers, prominent use of whitespace.
