A dedicated landing page for Orthopedic Specialists and Physiotherapists explaining the RecoverAI clinical tools, integration process, and data analytics dashboards.

**DESIGN SYSTEM (REQUIRED):**
Use Tailwind CSS.
Colors:
- `surface`: `#f5f0e8`
- `primary`: `#1a1a1a`
- `secondary`: `#e63b2e`
- `electric-lime`: `#C0F84B`
- `on-surface-variant`: `#4a4a4a`
- `surface-container-low`: `#f2ede5`
- `surface-container`: `#eee9e0`
Typography:
- Use `font-display` (Lato) for headings.
- Use `font-body` (Merriweather) for body text.
Style: Professional, medical, rounded-3xl containers, prominent use of whitespace.

**Page Structure:**
1. Header with navigation (consistent with index page)
2. Hero section focused on clinical precision and saving time
3. Feature breakdown of the Doctor Dashboard (AI Summaries, Protocol Assignment)
4. ROI Calculator or Data showing increased patient retention and recovery rates
5. Call to action to Request Demo or Clinic Onboarding
6. Footer with links (consistent with index page)
